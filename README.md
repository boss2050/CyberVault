# CyberVault

### Cryptographically Secure Password Security Suite

CyberVault is a modern cybersecurity-focused web application designed to analyze password strength, estimate crack resistance, detect breached/common passwords, and generate cryptographically secure passwords using the Web Cryptography API.

## 🔐 Features

### Password Security Analyzer

* Real-time password strength analysis
* Entropy calculation
* Offline crack-time estimation
* Security score visualization
* Dynamic security recommendations
* Visibility-on-screen warnings
* Breached/common password detection

### Cryptographically Secure Password Generator

* Uses `crypto.getRandomValues()` for secure randomness
* Adjustable password length
* Custom character set options
* Entropy + crack-time analysis for generated passwords
* One-click copy functionality

### Breach Detection System

* Simulated breach detection using real-world leaked password patterns
* RockYou + HaveIBeenPwned-inspired datasets
* Security recommendations for compromised passwords
* Animated breach scanning interface

## 🛡️ Security Features

* Local-only password analysis
* No password storage
* No external password transmission
* Cryptographically secure random password generation
* Real-time breach pattern detection
* Entropy-based security calculations


Run locally by opening `index.html` in your browser.
